Каркас плагина содержит комбобокс для слоев, пользователь может сам выбрать нужный слой, далее программа сама выполнит все расчеты.
В этой версии содержится обобщенный вариант метод для работы со слоями и фичами: добавление, редактирование, удаление, запросы и тд.
А также методы для записи результатов работы над слоями в шейпфайлы.
